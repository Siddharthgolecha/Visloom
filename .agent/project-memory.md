# Project memory

Durable, cross-task knowledge about Visloom. Human-curated, append-only.

This is **not** a scratchpad for the current task — that lives in
`.tasks/<id>/implementation.md`. Add entries here when a finding is worth
carrying across tasks: subtle invariants, calibration decisions, patterns
that keep biting, or judgement calls with a paper trail worth remembering.

Format: each entry is a `###` heading with a date and a short body citing
the file:line evidence.
